> Grapple for Speed.
> Speed Kills Enemies.
> Hesitate, Perish.

Try to keep moving for as long as possible as enemies swarm the arena trying to corner you. Use grapple points to swing at high speeds and crash into them - taking them out. How long can you survive?

## Controls

Arrow Keys - Movement

SPACE - Grapple
